二分搜尋法

二分搜尋法（Binary Search）是一種搜尋演算法，它的目的就是在一個排序好的數字中找到目標數字的位置。

原理

二分搜尋法的基本步驟如下：

  1. 確定中間元素：從已排序的數列中選取中間的元素。
  2. 比較目標值與中間元素：
    • 如果目標值等於中間元素，則搜尋成功。
    • 如果目標值小於中間元素，則繼續在中間元素左邊的區域搜尋。
    • 如果目標值大於中間元素，則繼續在中間元素右邊的區域搜尋。
  3. 重複步驟1和2，每次將搜尋範圍縮小一半，直到找到目標值或搜尋範圍變為空為止。

範例


def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = left + (right - left) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] > target:
            right = mid - 1
        else:
            left = mid + 1
    
    return -1  # 若找不到目標值，返回 -1

假設有已排序的數列 [1, 3, 5, 7, 9]，目標是找到數字7，演算法會先找到中間的數字5，然後繼續在右半邊搜尋，最終找到目標7。

優缺點

優點

二分搜尋法效率高，適合處理大型已排序的數列，時間複雜度為 O(log n)。

缺點

要求數列必須是已排序的，未排序的數列需要先進行排序操作。
