泡泡排序法

排序演算法有很多種，這一章我先介紹最簡單的 泡泡排序（Bubble Sort），幫助理解程式如何進行排序。

概念

泡泡排序的基本概念是，從數列的第一個數字開始，逐一與後面的數字進行比較。如果當前的數字比後面的數字大，就交換它們的順序。這個過程會反覆進行，直到所有數字都按照正確的順序排列。

範例


def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                # 交換元素
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

假設我們有數列 [3, 2, 1]，演算法會先比較 3 和 2，因為 2 比較小，兩者交換，結果變為 [2, 3, 1]。接著比較 3 和 1，交換後結果變為 [2, 1, 3]。最後再比較 2 和 1，交換後最終得到 [1, 2, 3]，這樣就完成了排序。

優缺點

優點

  • 容易理解，適合用來講解基礎排序演算法的運作方式。

缺點

  • 交換次數較多，這會導致時間複雜度增加。
  • 對於大型數據集來說，效率不高，時間複雜度為 O(n^2)。

結論

泡泡排序法通常用來講解排序的基本概念。由於其效率較低，實際應用中只適合處理小型數據集，對於較大的數據集，運行速度會非常慢。

心得

當我學習泡泡排序時，感覺它的實現方式非常簡單，讓我以為其他排序演算法也不會太難。但當我深入研究 Python 背後的排序演算法時，發現它們遠比泡泡排序複雜，這讓我感覺有種被泡泡排序「騙了」的感覺。
